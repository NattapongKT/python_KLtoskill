funtion คือ ชุดคำสั่งที่เราสร้างเอาเพียงครั้งเดียว เรียกใช้ได้รื่อยๆ ไม่ต้องเขียนใหม่ทุกครั้ง(สะดวกมาก)

parameter ไม่เท่ากับ argument
ตัวอย่าง
{
    def greet(name, greeting="Hello"): # name และ greeting คือ parameters
    print(f"{greeting}, {name}!")

    greet("Alice") # "Alice" คือ argument ที่ส่งให้ name
    greet("Bob", "Good morning") # "Bob" และ "Good morning" คือ arguments ที่ส่งให้ name และ greeting
}

return คือ คำสั่งที่ใช้ ออกจากฟังก์ชัน และ ส่งค่า (ผลลัพธ์) กลับไปยังส่วนของโค้ดที่เรียกฟังก์ชันนั้นๆ

ใน Python, Local variable (ตัวแปรโลคัล) คือตัวแปรที่สร้างและใช้ได้เฉพาะภายใน ฟังก์ชันหรือบล็อกโค้ดนั้นๆ เท่านั้น
(มีขอบเขตจำกัด) ส่วน Global variable (ตัวแปรโกลบอล) คือตัวแปรที่ประกาศไว้นอกฟังก์ชัน 
ทำให้สามารถเข้าถึงและใช้งานได้จากทุกส่วนของโปรแกรม (มีขอบเขตทั่วถึง) โดยการใช้ global keyword 
ในฟังก์ชันเพื่อเปลี่ยนแปลงค่าตัวแปรโกลบอลจากภายในฟังก์ชันได้.

สรุปความแตกต่าง
Local: มีชีวิตและใช้งานได้เฉพาะในฟังก์ชันที่ถูกสร้าง.
Global: มีอยู่ตลอดโปรแกรม เข้าถึงได้ทุกที่ และแชร์ข้อมูลระหว่างฟังก์ชันได้.
ข้อควรระวัง: การใช้ global มากเกินไปอาจทำให้โค้ดสับสนและเกิดข้อผิดพลาดได้.
